## 2025-05-15 - Improving Data Utility in Tables
**Learning:** Users often need to copy specific data points like stock tickers or transaction IDs from tables to use in other tools (e.g., brokerage apps, research sites). Providing a one-click copy mechanism significantly reduces friction compared to manual selection.
**Action:** Implement a "copy on hover" pattern for primary identifiers in table cells (tickers, IDs) using Lucide icons for feedback.
